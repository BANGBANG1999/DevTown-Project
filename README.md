# Sorting And Filtering

## Getting Started

These instructions will help you set up and run the project on your local machine.

### Prerequisites

- Node.js 
- npm

### Installing

1. Clone the repository:

   git clone https://github.com/BANGBANG1999/DevTown-Project.git
   
   cd DevTown-Project

   npm install

   npm run dev
